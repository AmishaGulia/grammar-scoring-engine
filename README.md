# Grammar Scoring Engine

## Overview
The **Grammar Scoring Engine** is a Python-based tool designed to **evaluate and score the grammatical quality of text**. It uses NLP techniques to provide a **numerical score** reflecting grammatical correctness. This engine can be applied in **essay scoring, writing assistance, or automated feedback systems**.

---

## Features
- Calculate grammar scores for text inputs  
- Provides detailed evaluation metrics:
  - **Mean Absolute Error (MAE)**  
  - **Root Mean Squared Error (RMSE)**  
  - **Pearson correlation**  
- Supports batch processing of text data  
- Can be integrated into web or desktop applications
